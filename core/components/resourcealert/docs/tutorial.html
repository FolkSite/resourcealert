Subscription type alerts for frontend users when resources are updated in CMS (subscribe to page update emails).

Installing resourcealert

Go to System | Package Management on the main menu in the MODX Manager and click on the "Download Extras" button. That will take you to the Revolution Repository (AKA Web Transport Facility). Put resourcealert in the search box and press Enter. Click on the "Download" button, and once the package is downloaded, click on the "Back to Package Manager" button. That should bring you back to your Package Management grid. Click on the "Install" button next to resourcealert in the grid. The resourcealert package should now be installed.
